📄 Document GPT

Document GPT is an AI-powered chatbot that allows users to upload documents and ask questions about their content.
The application uses a Retrieval-Augmented Generation (RAG) pipeline to retrieve relevant information from uploaded files and generate accurate responses.

🚀 Features

Chat with PDF, DOCX, and CSV files

Semantic search using vector embeddings

Context-aware conversation

Fast retrieval using FAISS

Simple Streamlit UI

🛠 Tech Stack

Python

Streamlit

LangChain

OpenAI API

FAISS

PyPDF2

python-docx

Pandas
